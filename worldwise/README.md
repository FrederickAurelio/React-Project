### 🤔 What Technology did I use
- useState
- useEffect
- useReducer
- React Router
- Context Api
- memo
- useMemo, useCallback
- Splitting Bundle Size

- Page Navigation
- Fake Login Authentication
- Record keeping for your travel
- Able to use current location
- Show list of traveled country

### 🏃‍♂️ How to run
- npm install
- npm run dev
- npm run server
